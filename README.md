# Project Scenario & Other Details
An analysis of a UK-based online store's customer transaction data

# Source: Coursera [https://www.coursera.org/projects/perform-exploratory-data-analysis-on-retail-data-with-python]

# Dataset: Online Retail.xslx

# Summary of Project Objectives:
1. Review data & perform any necessary cleaning
2. Analyze sales trend over time. Idenifty busiest sales months and days
3. Identify most valuable customers
4. Identify best-selling products based on quantity sold
5. Identify best-selling countries based on quantity sold
6. Identify dataset outliers/anomalies and discuss impact on analysis
7. Draw conclusions and make recommendations to improve sales performance and customer satisfaction
