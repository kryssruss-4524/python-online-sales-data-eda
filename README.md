# Project Scenario & Other Details

An analysis of a UK-based online store's customer transaction data
Source: Coursera [https://www.coursera.org/projects/perform-exploratory-data-analysis-on-retail-data-with-python]

Dataset: Online Retail.xslx

Summary of Project Objectives:
- Review data & perform any necessary cleaning
- Analyze sales trend over time. Idenifty busiest sales months and days
- Identify most valuable customers
- Identify best-selling products based on quantity sold
- Identify best-selling countries based on quantity sold
- Identify dataset outliers/anomalies and discuss impact on analysis
- Draw conclusions and make recommendations to improve sales performance and customer satisfaction
